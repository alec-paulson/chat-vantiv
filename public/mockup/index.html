Geoffrey

A js library that creates a chat window in a webpage and can answer simple questions.
built by @RealAlecPaulson and @iamandrewharris for Vantiv Hack Days - Fall 2017.

Put Geoffrey to Work

Every day as backoffice users we are constantly digging through mountains of data when what we all we really need is an intuitive way to interact with this data.

  • Show Me

How it Works

  • Say What?

    A lightweight javascript library that when pasted into a HTML page, creates an interactive experience that will answer your questions.

  • Really?

    Geoffrey works with CRM, Pricing, Activate, Click-to-Agree, you name it. Sure, even Sharepoint. Geoffrey also doesn't care what language your app is crafted in.

  • It's all about the data.

    Geoffrey connects to most databases and does the query for you.
    Geoffrey does the work, you take the credit.

  • Two core technologies for building Web pages. HTML provides the structure of the page, CSS the visual layout.


  • Relies on a stateless, client-server, cacheable communications protocol -- and in virtually all cases, the HTTP protocol is used.
